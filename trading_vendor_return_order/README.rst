============================
Trading Vendor Return Order
============================

The new module 'trading_vendor_return_order' The module 'purchase_order_return_stock_picking' inherits
model 'stock.picking', and links return stock picking order with purchase order by one additional field
'returned_purchase_id'. Users could check each return stock picking of related purchase order in tree view.
It allows user to generate the return picking order report for specific purchase order.
