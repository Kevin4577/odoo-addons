=======================
Report Py3o Multisheet
=======================

The module 'report_py3o_multisheet_custome_line' will inherit the module 'report_py3o_multisheet'.
The body of base template would be modified with several custome lines.

Installation
============
To install this module, you need to install ezodf python library:

sudo pip install ezodf
